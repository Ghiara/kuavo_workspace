  ************************************************************************

                                ID: 1       PROTOCOL: MODBUS                              

                                PORT: /dev/ttyUSB0       BAUDRATE: 115200

## Some info
- /dev/ttyUSB0 is left hand
- /dev/ttyUSB1 is right hand
